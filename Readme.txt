How to run this project?

Before going through the following steps  make sure you have an active internet connection

Step 1 > Install node
Step 2 > Open Command Line in the project directory
Step 3 > Enter "npm i" without qoutes and hit enter
Step 4 > Wait until npm installs the dependencies
Step 5 > Enter "node app" without qoutes to run the server
Step 6 > Open browser and go to address "http://localhost:3000/"
